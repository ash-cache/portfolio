    Speller

    A very fast spellchecker written in C.

    For reference, it takes 0.29 seconds to identify 1614 misspelled words in Pride and Prejudice which contains 125,203 words.

    It compares every word in a text against a hash table consisting of an array of linked lists.

    This was actually quite challenging. I researched hash functions and wrote a custom hash function to hash every word in an English dictionary.
